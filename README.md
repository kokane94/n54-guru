## 🏁 What is N54 Guru?

**N54 Guru** is a professional-grade Android diagnostic application specifically engineered for BMW N54 335i E93 convertibles. It provides real-time monitoring of all engine, transmission, electrical, and convertible roof systems with AI-powered predictive failure detection and multi-regional voice commentary.

### 🎯 Core Features

✅ **Real-time Engine Monitoring** - RPM, coolant temp, fuel pressure, ignition timing, boost pressure
✅ **Transmission Diagnostics** - Fluid temp, pressure, gear detection, shift analysis
✅ **Convertible Roof Control** - Full operation (open/close/tilt) with real-time status
✅ **Complete Electronics Monitoring** - Battery, lights, HVAC, steering, safety systems
✅ **AI Predictive Failure Detection** - Machine learning-based component failure prediction
✅ **12 Regional Voice Accents** - From Hori Maori to German Boss!
✅ **Live AI Voice Commentary** - Real-time explanations with recommended actions
✅ **Automatic Part Finder** - Searches AliExpress, Amazon, eBay for cheapest parts
✅ **Price Comparison** - Shows cheapest vs premium options with savings
✅ **Data Logging & Export** - 7-day rolling history with CSV export
✅ **Multi-View Display** - Circular gauges, health bars, raw data formats
✅ **OBD2 K-CAN Support** - Full USB adapter compatibility

## 🚀 Quick Start

### Prerequisites
- Android device (API 24+)
- USB OBD2 K-CAN adapter
- USB OTG adapter/cable
- BMW 335i E93

### Installation

```bash
git clone https://github.com/kokane94/n54-guru.git
cd n54-guru
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```
